# CodeDaily – 7-Day Python Challenge Plan

A simple, focused introduction to Python.
Each day includes one core concept and one hands-on challenge.

## Day 1 — Variables & Input

Concepts:

Variables

Basic data types (str, int, float, bool)

input()

print()

Challenge:
Write a program that asks for your name and age, then prints:
"Hello <name>, you are <age> years old."

## Day 2 — Conditionals

Concepts:

if / elif / else

Comparison operators

Challenge:
Write a program that takes a number and prints:

"Positive" if it is greater than zero

"Negative" if it is less than zero

"Zero" otherwise

## Day 3 — Lists

Concepts:

Creating lists

Indexing

Appending items

Getting list length

Challenge:
Create a list of five numbers.
Print:

The first number

The last number

The sum of all numbers

## Day 4 — Loops

Concepts:

for loops

Looping through ranges

Looping through lists

Challenge:
Print all even numbers between 1 and 50.

## Day 5 — Functions

Concepts:

Defining functions with def

Parameters

Return values

Challenge:
Create a function square(n) that returns the square of a number.
Test it with a few values.

## Day 6 — Strings

Concepts:

String slicing

.upper(), .lower(), .replace(), .split()

f-strings

Challenge:
Write a function that takes a sentence and returns it reversed
(e.g., "hello world" → "world hello").

## Day 7 — Dictionaries

Concepts:

Key/value pairs

Adding, reading, updating values

Iterating through a dictionary

Challenge:
Create a dictionary representing a user:

{
"name": "...",
"age": ...,
"city": "..."
}


Print each key and value on a separate line.
