# ✅ Simple 7-Day Beginner Challenge Plan
## Day 1 — Variables

Challenge:
Create 3 variables (name, age, isStudent) and print them.

## Day 2 — Functions

Challenge:
Write a function greet(name) that returns
"Hello, <name>!".

## Day 3 — Arrays

Challenge:
Given an array of numbers, return the sum.

## Day 4 — Conditionals

Challenge:
Given a number, return "even" or "odd".

## Day 5 — Loops

Challenge:
Print the numbers from 1 to 20.

## Day 6 — String Methods

Challenge:
Take a sentence and return it in uppercase.

## Day 7 — Objects

Challenge:
Create an object representing a book (title, author, pages) and print it.
